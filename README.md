# UBot - YouTube Audio Bot

Telegram-бот для скачивания аудио с YouTube. Конвертирует видео в MP3 и отправляет пользователям.

> MVP версия проекта

**Производительность и ресурсы:**
- Низкое потребление ОЗУ;
- Эффективная разбивка больших файлов на чанки;
- Быстрая обработка видео благодаря асинхронной архитектуре и очереди задач;


## Возможности

- Скачивание аудио с YouTube по ссылке
- Конвертация в MP3 (128 kbps)
- Автоматическая разбивка больших файлов на части
- Управление доступом администратором
- Очередь обработки запросов

## Установка

### Требования

- Python 3.11+
- FFmpeg

### Зависимости

```bash
pip install -r requirements.txt
```

### Переменные окружения

```env
BOT_TOKEN=
ADMIN_ID=
```

## Запуск

### Локально

```bash
python bot.py
```

### Docker Compose

```bash
docker-compose up -d
```
